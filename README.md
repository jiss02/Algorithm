### 익히면 좋을 함수 및 코드들

---

#### 특정 문자로 칸 채우기

1. zfill 함수
```python
    #"0002"
    "2".zfill(4)
```
2. rjust 함수
```python
    #00123
    "123".rjust(5,"0")
```

#### 비트연산자

파이썬은 기본적으로
```python
bin(), hex(), oct()
```
 함수를 제공한다.
 
| 연산자 | 기능 |
|:--|--:|
|&|비트간의 AND 연산|
|\||비트간의 OR 연산|
|^|비트간의 XOR 연산|
|~|비트간의 NOR 연산|
|<<|n번 비트를 왼쪽으로 이동시킴|
|>>|n번 비트를 오른쪽으로 이동시킴|
